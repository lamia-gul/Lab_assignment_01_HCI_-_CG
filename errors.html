HTML Error Section: Identifying Correct Semantic Use

10 Common HTML & Accessibility Errors

Correcting Common Mistakes
# Error Description Wrong Code Corrected Code
1 Link directly in UL <ul><a>...</a></ul> <ul><li><a>...</a></li></ul>
2 Multiple H1 Tags Two <h1> on one page Exactly one <h1> per page
3 No Label Association <input id="n"> Name <label for="n">Name</label><input id="n">
4 Skipping Headings <h1> then <h3> <h1> then <h2>
5 Missing Alt Text <img src="x.jpg"> <img src="x.jpg" alt="Description">
6 Vague Links <a>Click Here</a> <a>Download Dix Ch. 12</a>
7 No Table Caption <table>...</table> <table><caption>Info</caption>...</table>
8 Table for Layout Using <table> for grid alignment Use <section> or <article>
9 Div Soup Using <div> for headers/nav Using <header>, <nav>
10 Misplaced Nav <nav> outside <body> <nav> inside <body>
